
# Finance Sales & Profit Power BI Dashboard

## Project Overview
This project presents a Finance Sales and Profit Dashboard developed using Power BI to analyze financial performance, sales trends, and profitability metrics.

## Objectives
- Analyze overall sales performance
- Evaluate profit trends
- Identify high-performing products and regions
- Track key financial KPIs

## Dataset
The dataset includes financial sales transactions containing revenue, profit, product, and regional information.

## Tools Used
- Power BI
- Excel
- Data Analysis
- Data Visualization

## Dashboard Features
- Sales and Profit KPI cards
- Trend analysis charts
- Category and regional performance analysis
- Interactive dashboard visuals
